# 🧭 Me Project Dashboard

> **Usage**: `/menu` - 顯示可用指令總覽。

## 🚀 Core (核心指令)

| 指令      | 說明                         |
| :-------- | :--------------------------- |
| `/run`    | 啟動開發伺服器 (npm run dev) |
| `/commit` | 提交程式碼變更 (需審查)      |
| `/deploy` | 部署至 GitHub Pages          |
| `/stop`   | 停止所有開發伺服器           |

## 🧠 Brain (AI 協作)

| 指令       | 說明                               |
| :--------- | :--------------------------------- |
| `/advisor` | 啟動顧問團隊 (PM/CTO/Designer/CSO) |
| `/todo`    | 管理待辦事項與 Backlog             |
| `/journal` | 檢視每日工作日誌                   |
| `/cleanup` | 清理專案架構 (歸檔舊檔)            |

## 🛠️ Tools (實用工具)

| 指令            | 說明                       |
| :-------------- | :------------------------- |
| `/new-trip`     | 建立新旅程頁面 (React App) |
| `/test`         | 執行自動化測試             |
| `/check-change` | 檢查 Git 狀態與變更        |

## ❓ Help (求助)

| 指令    | 說明                         |
| :------ | :--------------------------- |
| `/help` | 呼叫專家 (PM, CTO, Designer) |
