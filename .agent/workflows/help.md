# Help Workflow (Me)

> **需要導航？** 請輸入 `/menu` 查看指令總覽。
> **需要專家？** 請使用以下指令直接呼叫。

## 👥 呼叫專家 (Call Experts)

| 指令                    | 用途                                         |
| :---------------------- | :------------------------------------------- |
| `/help pm [問題]`       | **產品經理**: 詢問進度、功能優先序、需求澄清 |
| `/help cto [問題]`      | **技術長**: 詢問架構、重構建議、技術選型     |
| `/help designer [問題]` | **設計師**: 詢問 UI/UX、配色、版面配置       |

## 🤖 自動分診

若不確定該問誰，直接輸入 `/help [您的問題]`，AI 接待員會為您分派。
