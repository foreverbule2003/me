# 網站功能清單

> 最後更新：2024-12-22

## 網站概覽

| 項目         | 說明                                               |
| ------------ | -------------------------------------------------- |
| **網站主題** | TimZ - POCKET DEVICE（復古 Game Boy 風格個人網站） |
| **視覺風格** | 像素風格、復古電玩配色（綠色調螢幕）               |
| **技術棧**   | HTML, CSS, JavaScript, TailwindCSS, React (CDN)    |

---

## 頁面結構

```
my-landing-page/
├── index.html          # 首頁（TimBoy 模擬器）
├── about.html          # 關於我
├── trips.html          # 旅程列表
├── tools.html          # 工具箱
├── trips/
│   ├── 2025-cebu/      # 2025 宿霧
│   ├── 2025-osaka/     # 2025 大阪
│   ├── 2026-hokkaido/  # 2026 北海道
│   ├── 2026-ise-shima/ # 2026 伊勢志摩
│   └── shared/         # 共用元件
└── tools/
    ├── bull-put-spread.html      # 期權策略模擬器
    └── financial-dashboard.html  # 財務儀表板
```

## 功能詳情

### 1. 首頁 (index.html)

| 功能              | 說明                                |
| ----------------- | ----------------------------------- |
| **TimBoy 模擬器** | 模擬 Game Boy 外觀的互動介面        |
| **開機動畫**      | 點擊 Power On 後的啟動動畫效果      |
| **音效系統**      | 復古電玩音效（可開關）              |
| **D-Pad 導航**    | 使用虛擬方向鍵或鍵盤上下鍵瀏覽選單  |
| **A/B 按鈕**      | A 按鈕確認選擇、進入頁面            |
| **選單項目**      | ABOUT、SELECT TRIP、TOOLS & CONTACT |

### 2. 關於頁 (about.html)

| 功能         | 說明               |
| ------------ | ------------------ |
| **個人簡介** | 展示個人背景與興趣 |
| **返回按鈕** | 回到主選單         |

### 3. 旅程頁 (trips.html)

| 功能         | 說明                         |
| ------------ | ---------------------------- |
| **旅程列表** | 展示所有已規劃的旅程         |
| **旅程卡片** | 每個旅程顯示名稱、日期、狀態 |

#### 3.1 旅程詳情頁（以 2026-ise-shima 為例）

| 功能              | 說明                                                       |
| ----------------- | ---------------------------------------------------------- |
| **分頁導航系統**  | 五個分頁：行程總覽、每日詳情、行程地圖、美食指南、預算規劃 |
| **Sticky 分頁列** | 分頁導航固定在頂部，滾動時保持可見                         |
| **響應式分頁**    | 手機版只顯示圖示，桌面版顯示圖示+文字                      |

#### 3.2 行程總覽分頁

| 功能             | 說明                                             |
| ---------------- | ------------------------------------------------ |
| **行程亮點區塊** | 統計卡片（天數、周遊券天數、素食友善、溫泉療癒） |
| **階段總覽卡片** | 顯示兩個階段的每日標題摘要                       |
| **實用連結區塊** | 分類展示交通票券、住宿、景點的外部連結           |

#### 3.3 每日詳情分頁

| 功能               | 說明                                    |
| ------------------ | --------------------------------------- |
| **可折疊階段標題** | 點擊階段標題可展開/折疊該階段的所有行程 |
| **可折疊 DayCard** | 點擊卡片圖片可展開/折疊詳細內容         |
| **全域展開/折疊**  | FAB 按鈕可一鍵展開/折疊所有卡片         |
| **時間軸活動列表** | 按時間順序列出每日活動                  |
| **地圖/交通圖示**  | 點擊活動項目的圖示可開啟 Google Maps    |
| **美食連結**       | 餐飲活動可直接跳轉到美食指南分頁        |
| **警告/資訊提示**  | 橘色警告框和藍色資訊框                  |
| **每日亮點**       | 卡片底部的 Highlight 區塊               |
| **回到頂部按鈕**   | 頁面底部的滾動回頂按鈕                  |

#### 3.4 行程地圖分頁

| 功能               | 說明                                   |
| ------------------ | -------------------------------------- |
| **近鐵特急比較表** | 特急 vs 普通列車的時間、費用比較表     |
| **巴士時刻表**     | 松阪⇔VISON 雙向時刻表                  |
| **每日路徑總覽**   | 10天的關鍵交通路線卡片，可點擊開啟地圖 |
| **可折疊區塊**     | 每個區塊可獨立展開/折疊                |

#### 3.5 美食指南分頁

| 功能                 | 說明                                       |
| -------------------- | ------------------------------------------ |
| **按地區分類**       | 臨空城、VISON、伊勢、賢島、大阪梅田、USJ   |
| **可折疊分類**       | 點擊地區標題可展開/折疊                    |
| **愛心收藏功能**     | 點擊愛心收藏餐廳，使用 localStorage 持久化 |
| **收藏排序**         | 已收藏項目自動排序至該分類頂部             |
| **推薦標籤**         | 推薦餐廳顯示高亮標籤                       |
| **Google Maps 連結** | 每個餐廳可直接開啟地圖導航                 |

#### 3.6 預算規劃分頁

| 功能             | 說明                               |
| ---------------- | ---------------------------------- |
| **響應式表格**   | 桌面版顯示表格，手機版顯示卡片列表 |
| **雙幣別顯示**   | 同時顯示 JPY 和 TWD 預估金額       |
| **自動計算總計** | 自動加總所有項目                   |
| **預訂建議提示** | 底部的資訊提示區塊                 |

#### 3.7 Google Maps 整合

| 功能           | 說明                          |
| -------------- | ----------------------------- |
| **嵌入式地圖** | Modal 內嵌 Google Maps iframe |
| **路線模式**   | 顯示兩點之間的交通路線        |
| **定點模式**   | 顯示單一地點位置              |
| **外部連結**   | 可在 Google Maps App 中開啟   |

#### 3.8 AI 旅遊助手

| 功能             | 說明                                   |
| ---------------- | -------------------------------------- |
| **FAB 入口**     | 右下角浮動按鈕開啟                     |
| **行程顧問對話** | AI 聊天功能（支援 Gemini API）         |
| **素食溝通卡**   | 預設的日文素食需求卡片，可直接給店員看 |
| **飲食禁忌說明** | NG 紅區/OK 綠區清楚標示                |
| **高湯確認語句** | 預設的日文詢問語句                     |

### 4. 工具頁 (tools.html)

| 功能         | 說明               |
| ------------ | ------------------ |
| **工具列表** | 展示可用的工具集合 |
| **聯絡資訊** | Email 連結         |

#### 4.1 期權策略模擬器 (bull-put-spread.html)

| 功能         | 說明                           |
| ------------ | ------------------------------ |
| **參數輸入** | 標的價格、履約價、權利金       |
| **即時計算** | 最大獲利、最大虧損、損益兩平點 |
| **風險指標** | 風險回報比視覺化               |
| **損益圖表** | 動態繪製到期損益曲線           |

#### 4.2 財務儀表板 (financial-dashboard.html)

| 功能             | 說明                 |
| ---------------- | -------------------- |
| **營收趨勢圖**   | 顯示營業收入變化     |
| **獲利能力圖**   | 長條圖展示獲利指標   |
| **時間區間切換** | 季度 / 半年度 / 年度 |
| **互動圖表**     | 滑鼠懸停顯示詳細數據 |

## 資料結構

### itineraryData (行程資料)
```javascript
[
  {
    phase: "第一階段：伊勢志摩度假慢旅 (Day 1-6)",
    days: [
      {
        day: 1,
        date: "1/12 (一)",
        title: "抵達關西 · Rinku Outlet",
        image: "https://...",
        time: "13:00 - 住宿",
        activities: [{ time, text, subText, note, tips, map, foodGuideLink }],
        highlight: "🛍️ 落地即購物..."
      }
    ]
  }
]
```

### budgetData (預算資料)
```javascript
[
  { item: "機票 (TPE-KIX)", cost: 42000, note: "說明" }
]
```

### foodData (美食資料)
```javascript
{
  categories: [
    {
      location: "VISON 園區",
      day: "Day 2-3",
      sections: [
        {
          title: "🥗 素食友善餐廳",
          items: [{ name, type, desc, note, recommended, mapUrl }]
        }
      ]
    }
  ]
}
```
