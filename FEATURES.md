# 網站功能清單

> 最後更新：2024-12-22

## 網站概覽

| 項目         | 說明                                               |
| ------------ | -------------------------------------------------- |
| **網站主題** | TimZ - POCKET DEVICE（復古 Game Boy 風格個人網站） |
| **視覺風格** | 像素風格、復古電玩配色（綠色調螢幕）               |
| **技術棧**   | HTML, CSS, JavaScript, TailwindCSS                 |

---

## 頁面結構

```
my-landing-page/
├── index.html          # 首頁（TimBoy 模擬器）
├── about.html          # 關於我
├── trips.html          # 旅程列表
├── tools.html          # 工具箱
├── trips/
│   ├── 2025-cebu/      # 2025 宿霧
│   ├── 2025-osaka/     # 2025 大阪
│   ├── 2026-hokkaido/  # 2026 北海道
│   ├── 2026-ise-shima/ # 2026 伊勢志摩
│   └── shared/         # 共用元件
└── tools/
    ├── bull-put-spread.html      # 期權策略模擬器
    └── financial-dashboard.html  # 財務儀表板
```

## 功能詳情

### 1. 首頁 (index.html)

| 功能              | 說明                                |
| ----------------- | ----------------------------------- |
| **TimBoy 模擬器** | 模擬 Game Boy 外觀的互動介面        |
| **開機動畫**      | 點擊 Power On 後的啟動動畫效果      |
| **音效系統**      | 復古電玩音效（可開關）              |
| **D-Pad 導航**    | 使用虛擬方向鍵或鍵盤上下鍵瀏覽選單  |
| **A/B 按鈕**      | A 按鈕確認選擇、進入頁面            |
| **選單項目**      | ABOUT、SELECT TRIP、TOOLS & CONTACT |

### 2. 關於頁 (about.html)

| 功能         | 說明               |
| ------------ | ------------------ |
| **個人簡介** | 展示個人背景與興趣 |
| **返回按鈕** | 回到主選單         |

### 3. 旅程頁 (trips.html)

| 功能         | 說明                         |
| ------------ | ---------------------------- |
| **旅程列表** | 展示所有已規劃的旅程         |
| **旅程卡片** | 每個旅程顯示名稱、日期、狀態 |

#### 3.1 旅程詳情頁（以 2026-ise-shima 為例）

| 功能           | 說明                                             |
| -------------- | ------------------------------------------------ |
| **分頁系統**   | 行程總覽、每日詳情、行程地圖、美食指南、預算規劃 |
| **每日時間軸** | 精確列出每日景點、交通與住宿                     |
| **素食標註**   | 標註適合素食者的餐廳與選項                       |
| **住宿資訊**   | 飯店名稱、地址、特色說明                         |
| **交通資訊**   | 詳細交通方式與時間                               |

### 4. 工具頁 (tools.html)

| 功能         | 說明               |
| ------------ | ------------------ |
| **工具列表** | 展示可用的工具集合 |
| **聯絡資訊** | Email 連結         |

#### 4.1 期權策略模擬器 (bull-put-spread.html)

| 功能         | 說明                           |
| ------------ | ------------------------------ |
| **參數輸入** | 標的價格、履約價、權利金       |
| **即時計算** | 最大獲利、最大虧損、損益兩平點 |
| **風險指標** | 風險回報比視覺化               |
| **損益圖表** | 動態繪製到期損益曲線           |

#### 4.2 財務儀表板 (financial-dashboard.html)

| 功能             | 說明                 |
| ---------------- | -------------------- |
| **營收趨勢圖**   | 顯示營業收入變化     |
| **獲利能力圖**   | 長條圖展示獲利指標   |
| **時間區間切換** | 季度 / 半年度 / 年度 |
| **互動圖表**     | 滑鼠懸停顯示詳細數據 |

---

## 互動元素

| 元素         | 位置       | 說明                   |
| ------------ | ---------- | ---------------------- |
| **音效切換** | 首頁右上角 | Sound On/Off           |
| **鍵盤支援** | 全站       | 上下鍵導航、Enter 確認 |
| **按鈕回饋** | 全站       | 點擊時有視覺/音效回饋  |
| **圖表互動** | 工具頁     | 滑鼠懸停顯示數據       |
