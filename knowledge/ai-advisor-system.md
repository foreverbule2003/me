# 🤖 AI 顧問系統 (AI Advisor System)

這是本專案的核心架構之一：一個「層級式 AI 專家團隊」，用於協助專案管理與設計決策。

---

## 🧠 設計理念

**問題**：單一 AI 視角容易有盲點，且不同任務需要不同專長。

**解決方案**：建立多角色協作系統，各司其職。

---

## 👥 專家團隊

| 角色     | 檔案                  | 專長                             |
| -------- | --------------------- | -------------------------------- |
| 🎯 接待員 | `receptionist-prompt` | 分診、引導使用者找到正確的專家   |
| 🛠️ PM     | `web-pm-prompt`       | 進度管理、Backlog 排序、UX 檢視  |
| 🔧 技術長 | `tech-lead-prompt`    | 架構設計、程式碼品質、效能優化   |
| 🎨 設計師 | `designer-prompt`     | UI/UX 設計、視覺美感、使用者體驗 |

---

## 🔗 協作流程

```
/help [描述問題]
         │
         ▼
┌─────────────────────────┐
│  接待員 (分診)          │ ← 不確定時先來這
└─────────────────────────┘
         │
         ▼
┌─────────────────────────┐
│  專家 (PM/Tech/Design)  │ ← 專業視角分析
└─────────────────────────┘
         │
         ▼
      解決方案
```

---

## 🎯 使用方式

| 情境               | 指令               |
| ------------------ | ------------------ |
| 不確定該找誰       | `/help [描述問題]` |
| 產品規劃、進度管理 | `/pm`              |
| 技術架構、效能問題 | `/tech`            |
| UI/UX 設計         | `/design`          |
